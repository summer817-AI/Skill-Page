aihot

AI 资讯 推荐：高 信息抓取 研究分析 作者：KKKKhazix 来源：GitHub

让 agent 直接查询 AI HOT 的公开 API，生成中文 AI 资讯简报，不需要 API Key 或 MCP server。

适合场景

  • 用户问“今天 AI 圈有什么”“最近 OpenAI/Anthropic/Google 发了什么”。
  • 需要基于实时资讯生成日报、精选条目、公司或关键词动态。

输入输出

  • 输入：自然语言问题、日期、关键词、分类或时间窗口。
  • 输出：中文 markdown 简报、精选条目、来源链接和摘要。

安装方法

Windows PowerShell
git clone https://github.com/KKKKhazix/khazix-skills.git
New-Item -ItemType Directory -Force "$env:USERPROFILE\.codex\skills" | Out-Null
Copy-Item -Recurse .\khazix-skills\aihot "$env:USERPROFILE\.codex\skills\aihot"

macOS / Linux
git clone https://github.com/KKKKhazix/khazix-skills.git
mkdir -p ~/.codex/skills
cp -R khazix-skills/aihot ~/.codex/skills/aihot

career-skill-planner

职业 Skill 规划 推荐：中高 产品PM 研究分析 作者：zephyrwang6 来源：GitHub

输入职业名称或 JD，拆解高频工作流，并输出每个工作流可制作成 skill 的提纲和提示词。

适合场景

  • 用户想知道某个职业如何用 AI 提效，或要为岗位打造专属 skill 库。
  • 从“我是设计师/运营/销售/律师”拆出可落地的 agent 工作流。

输入输出

  • 输入：职业、JD、核心输出物、最耗时的工作环节。
  • 输出：工作流拆解、候选 skill 清单、制作提示词和实现逻辑。

安装方法

Windows PowerShell
git clone https://github.com/zephyrwang6/career.skill.git
New-Item -ItemType Directory -Force "$env:USERPROFILE\.codex\skills" | Out-Null
Copy-Item -Recurse .\career.skill "$env:USERPROFILE\.codex\skills\career.skill"

macOS / Linux
git clone https://github.com/zephyrwang6/career.skill.git
mkdir -p ~/.codex/skills
cp -R career.skill ~/.codex/skills/career.skill

调用指令

Codex

SPACE-product-describer

产品描述 推荐：中高 产品PM 文字处理 开发工作流 作者：zephyrwang6 来源：GitHub

用六维产品描述法把模糊产品想法变成 agent 或开发团队能直接动手的结构化描述。

适合场景

  • 用户说“我想做一个产品/工具/应用”，但描述太笼统。
  • 写 PRD 或做原型前，需要先补齐用户流程、场景、边界和切入建议。

输入输出

  • 输入：产品想法、目标用户、流程、约束、期望原型或代码产物。
  • 输出：结构化产品描述、澄清问题和推荐切入角度。

安装方法

Windows PowerShell
git clone https://github.com/zephyrwang6/pm-product-describer.git
New-Item -ItemType Directory -Force "$env:USERPROFILE\.codex\skills" | Out-Null
Copy-Item -Recurse .\pm-product-describer "$env:USERPROFILE\.codex\skills\pm-product-describer"

macOS / Linux
git clone https://github.com/zephyrwang6/pm-product-describer.git
mkdir -p ~/.codex/skills
cp -R pm-product-describer ~/.codex/skills/pm-product-describer
